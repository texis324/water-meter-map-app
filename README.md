# 水道検針ルートマップ

検針ルートを地図上で管理する個人用Webアプリ。

## 機能
- ピン追加/編集/ドラッグ移動、並べ替え、なぞり順、連結、グループ化
- ルート線描画（Google Directions）、スタンプ、範囲削除、Undo/Redo
- リファレンスピン（薄ピン）への磁石スナップ
- 欠番・重複検出、スマホ対応

## データについて
検針データ（住民の氏名・住所等の個人情報）は**このリポジトリには含みません**。
アプリ起動後にローカルのJSONを「📥読込み」して使用し、データはブラウザのlocalStorageにのみ保存されます。

## 技術
Leaflet.js + OpenStreetMap/CartoDBタイル + Google Directions/Geocoding API（リファラー制限済み）
